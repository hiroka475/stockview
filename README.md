# StockView v1.0.0

日本株・米国株のチャート分析Webアプリです。
TradingViewライクなローソク足チャートで銘柄分析ができます。

## 主な機能

- ローソク足チャート（日足・週足対応）
- テクニカル指標（EMA200、ボリンジャーバンド ±2σ）
- トレンドライン描画（延長線を両方向に描画、クリップ対応）
- ウォッチリスト（JP/US/カスタムタブ対応、ドラッグ&ドロップ並び替え）
- 配当情報の表示（配当利回り、年間配当金、次回配当落ち日）
- ファンダメンタルズ分析（業績推移、営業利益率、配当推移など）
- スクリーニング機能（高配当株の条件検索）
- 価格アラート設定
- VIX / 米国10年債利回り / ドル円にも対応
- 日本株・米国株の両方に対応（日本株3,000+銘柄、米国株140+銘柄）

## ローカルで使う場合

### 必要なもの

- Python 3.9 以上

### 起動方法

1. 必要なパッケージをインストール（初回のみ）

```bash
pip3 install -r requirements.txt
```

2. サーバーを起動

```bash
python3 server.py
```

3. ブラウザで http://localhost:8080 にアクセス

4. 停止するときは `Ctrl + C`

## Renderにデプロイする方法

1. このリポジトリをGitHubにpush
2. [Render](https://render.com) でアカウント作成（無料）
3. 「New → Web Service」からGitHubリポジトリを接続
4. 設定は `render.yaml` で自動検出されます
5. デプロイ完了後、表示されるURLにアクセス

## 使い方

- **銘柄検索**: 右上の検索欄にコードまたは名前を入力（例: `7203`, `AAPL`, `トヨタ`, `VIX`）
- **ウォッチリスト**: 検索結果の「+」で追加。タブでJP/US/カスタムリストを切り替え
- **チャート操作**: マウスドラッグでスクロール、ホイールでズーム
- **期間変更**: 「1Y」「3Y」「5Y」「10Y」「15Y」「MAX」ボタン
- **日足/週足**: ツールバーのボタンで切り替え
- **トレンドライン**: 「延長線」ボタンでチャート上に2点クリックで描画
- **ファンダメンタルズ**: 銘柄選択後「ファンダメンタルズ分析」ボタン

## ファイル構成

```

├── server.py                ... APIサーバー（Yahoo Finance / IR BANKからデータ取得）
├── index.html               ... メイン画面（チャート・ウォッチリスト・詳細パネル）
├── fundamentals.html        ... ファンダメンタルズ分析画面
├── screening.html           ... スクリーニング画面
├── jp-stocks.json           ... 日本株マスタデータ（3,000+銘柄）
├── us-stocks.json           ... 米国株マスタデータ（140+銘柄）
├── dividend-candidates.json ... 高配当株スクリーニング候補データ
├── expand-stocks.py         ... マスタデータ拡張スクリプト
├── update-screening-data.py ... スクリーニングデータ更新スクリプト
├── requirements.txt         ... Python依存パッケージ
├── render.yaml              ... Renderデプロイ設定
├── manifest.json            ... PWA設定
├── sw.js                    ... Service Worker
└── README.md                ... このファイル
```

## データソース

- 株価データ: Yahoo Finance（15〜20分の遅延あり）
- ファンダメンタルズ: IR BANK（日本株のみ）
