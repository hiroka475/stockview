# StockView - 株式チャートWebアプリ

TradingViewライクなローソク足チャートで日本株・米国株を表示するWebアプリです。

## 起動方法（Mac）

### 1. ターミナルを開く

Mac の「アプリケーション」→「ユーティリティ」→「ターミナル」を開きます。

### 2. stock-view フォルダに移動

```bash
cd ~/Documents/Claude/claude-code-lab/stock-view
```

（フォルダの場所が違う場合は適宜変更してください）

### 3. サーバーを起動

```bash
python3 server.py
```

以下のように表示されたら起動完了です：

```
==================================================
  StockView サーバー起動
  http://localhost:8080
==================================================
  Ctrl+C で停止できます
```

### 4. ブラウザで開く

Safari または Chrome で以下にアクセスしてください：

**http://localhost:8080**

### 5. 停止する

ターミナルで `Ctrl + C` を押すとサーバーが停止します。

## 使い方

- **銘柄切り替え**: 右上の検索欄にティッカーシンボルを入力（例: `AAPL`, `7203`）
- **時間足の変更**: 「日足」「週足」「月足」ボタンをクリック
- **期間の変更**: 「1M」「3M」「6M」「1Y」「5Y」ボタンをクリック
- **チャート操作**: マウスドラッグでスクロール、ホイールでズーム
- **詳細情報**: チャート上にマウスを合わせると始値・高値・安値・終値・出来高を表示

## ファイル構成

```
stock-view/
├── server.py      ... Python製のAPIサーバー（Yahoo Financeからデータ取得）
├── index.html     ... フロントエンド（チャート表示・操作UI）
└── README.md      ... このファイル
```

## 注意事項

- Python 3 が必要です（Macには標準でインストールされています）
- 株価データはYahoo Financeから取得します（15〜20分の遅延あり）
- サーバーに接続できない場合はサンプルデータが表示されます
